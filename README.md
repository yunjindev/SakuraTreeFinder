# 🌸 Sakura Tree Finder

Find the nearest cherry blossom tree to your location.

**Live site:** [sakura-tree-finder.vercel.app](https://sakura-tree-finder.vercel.app/)

## What It Does

Sakura Tree Finder uses your device's GPS to locate the closest cherry blossom (sakura) tree near you. It also includes an interactive map where you can browse and search for sakura trees anywhere in the world.

## Features

- **Nearest tree locator** — tap a button, share your location, and get the closest sakura tree with species info, distance, and a link to open directions in your maps app.
- **Interactive map** — browse pink-dotted sakura trees on a full map with search functionality.
- **Reference guide** — common sakura species and their bloom periods.

## Data Sources

- Tree data from [OpenStreetMap](https://www.openstreetmap.org) contributors (ODbL)
- Map tiles from [OpenFreeMap](https://openfreemap.org)
- Rendered with [MapLibre](https://maplibre.org)
